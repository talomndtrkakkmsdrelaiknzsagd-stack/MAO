# 五升六暑假学习台发布说明

这个目录是可发布的网站根目录。

- `index.html`：学习台主页面

推荐发布方式：

1. GitHub Pages：把本目录内容上传到公开仓库，开启 Pages，来源选择 `main` 分支根目录。
2. Netlify：新建站点，直接拖入本目录。
3. Cloudflare Pages：新建 Pages 项目，上传本目录或连接 Git 仓库。

注意：

- 页面本身不需要服务器，属于纯静态网页。
- 孩子的答题、打卡、批改记录保存在当前浏览器本地，不会自动同步到互联网。
- 如果换设备访问，之前的本地记录不会自动出现。
- 发布后任何拿到链接的人都能打开学习台，请不要在页面里写真实姓名、手机号、学校等隐私信息。
